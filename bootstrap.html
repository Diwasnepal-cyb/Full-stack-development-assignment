What is Bootstrap?

This is a morden responsive site built using BootStrap.
lets understand what is BootStrap.

Bootstrap is an open-source, free front-end framework, used for building websites and web applications. It is used to create mobile-first responsive design websites and provides a complete set of syntax for template designs.
As a framework, Bootstarp includes the basics for responsive web development, so developers only need to inserts the code into the pre-define grid system. The Bootstrap framework is built on HTML, cascading style sheets CSS and JavaScript.

1. What is Bootstrap CSS?

  The most popular CSS framework for developing responsive and mobile-first websites is Bootstrap. The newest version is Bootstrap 5.

Why To Use Bootstrap?

  • ✔ Responsive design for all devices
  • ✔ Pre-built components like navbar, buttons, and forms
  • ✔ Faster development and easy customization
  • ✔ Consistent design across all browsers

- What is Bootstrap Protocol?

Bootstrap Protocol (BOOTP) is an internet protocol where a network client can be automatically configured to get an IP address and an OS booted with no user intervention. The BOOTP server is controlled by a network administrator and automatically assigns the IP address from a pool of addresses for a limited period of time.
