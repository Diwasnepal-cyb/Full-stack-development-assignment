About the Site

In the file index.html, I started building the basic structure of my portfolio website using HTML. This file includes the header section, which has the website title coupled with a brief description relevant to full stack development. In addition, I have included navigation hyperlinks to support user movement between different pages, namely About, Projects, and Contact. The Bootstrap layout is used by including a reference to its CDN, thus enabling efficient styling, and I have also included a reference to my custom CSS file to improve the visual presentation of the layout design. In addition, there is a reference to a JavaScript file to enable the possibility of incorporating future interactive elements, like form validation or dynamic effects. The major point of this page is to define the purpose of the site while at the same time presenting a good first impression of my work.
