What is JavaScript?

JavaScript is a programming or scripting language that enables you to add advanced features to web pages — whenever a web page does more than just sit there and show you static information for you to read — showing dynamic content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc. — you can be sure that JavaScript is likely being used. It is the third layer of the layer cake of standard web technologies, two of which (HTML and CSS) we have discussed in far greater detail elsewhere in the Learning Area.

Lets learn this with figure...

  • HTML: HTML is the markup language used for structuring and conveying meaning to web content, including specifying paragraphs, headings, and data tables, and linking images and videos in a web page.
  • CSS: CSS is a styling language consisting of rules that we use to add styling to our HTML content, such as background colors and fonts, and positioning our content in multiple columns.
  • JavaScript: JavaScript is a script language that allows you to implement dynamically updating content, control multimedia, animate images, and just about anything else. (Alright, not anything, but it is remarkable what can be done with some lines of JavaScript code.)
